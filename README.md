# Трибунал Луту

Фентезійний калькулятор розподілу Albion silver для данж-паті.

[Відкрити застосунок](https://yuroky.github.io/AlbionLootCalculator/)

## Про проєкт

Трибунал Луту допомагає групі з 2, 3 або 5 учасників рівномірно поділити лут після данжу. Введи вартість луту кожного гравця в Albion silver, і застосунок покаже, хто кому скільки має віддати.

Інтерфейс зроблений у стилі Albion-подібного market/inventory вікна: пергаментні панелі, дерев'яно-металеві рамки, silver-акценти й компактна таблиця розрахунку.

## Можливості

- Групи на 2, 3 і 5 учасників
- Введення Albion silver з перевіркою цілих чисел
- Автоматичний розрахунок рівної частки
- Зрозумілий список переказів між учасниками
- Обробка остачі, якщо загальна сума не ділиться рівно
- Адаптивний інтерфейс для десктопу й мобільних
- Автоматичний деплой на GitHub Pages

## Технології

- Vue 3
- Vite
- Vitest
- Lucide icons

## Локальний запуск

```bash
npm install
npm run dev
```

Запуск тестів:

```bash
npm run test
```

Локальна збірка:

```bash
npm run build
```

Збірка для GitHub Pages:

```bash
npm run build:pages
```

## Деплой

Застосунок деплоїться на GitHub Pages через `.github/workflows/deploy-pages.yml`.
